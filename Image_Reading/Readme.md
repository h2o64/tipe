Documentation : http://www.fil.univ-lille1.fr/~wegrzyno/portail/API1/Doc/TP/TP-Images/Doc/Images.html

Compile the module with :
```shell
> ocamlc -c images.mli
> ocamlc -c images.ml
```

Open the libraries in oCAML instance:
```shell
> ocaml graphics.cma images.cmo
```
